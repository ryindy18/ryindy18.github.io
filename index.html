This is a modal factoid

This modal included a span tag with the ampersand symbol joined with the word times within it to visually recreate a close button 'x' to exit the pop up. It is simply not written as an 'x' so as to not confuse those who use screen readers to read it as the literal letter.

They are useful for web development because it can provide an effect without reloading the page or using a traditional window alert box.
